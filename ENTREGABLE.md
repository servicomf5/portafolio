## 4. Descripción del Proyecto

Este portafolio profesional fue desarrollado como proyecto final del Bootcamp de Python, demostrando competencias en:

### Tecnologías Utilizadas
- **Backend**: Django 5.x (Python 3.12)
- **Base de Datos**: PostgreSQL 17
- **Frontend**: Bootstrap 4, jQuery, HTML5, CSS3
- **Contenedores**: Docker + Docker Compose
- **CI/CD**: GitHub Actions
- **Deployment**: VPS (Contabo)

### Funcionalidades Implementadas

1. **Perfil Profesional**
   - Información personal dinámica desde base de datos
   - Foto de perfil
   - Título profesional y descripción

2. **Sección Skills**
   - Habilidades técnicas con porcentajes visuales
   - Habilidades blandas

3. **Experiencia Laboral**
   - Timeline de experiencia profesional
   - Educación y certificaciones

4. **Portafolio de Proyectos**
   - Galerías de proyectos
   - Lightbox para visualización
   - Imágenes y descripciones

5. **Caso de Estudio**
   - Desafío identificado
   - Solución propuesta
   - Herramientas utilizadas
   - Métricas de impacto
   - Aprendizajes

6. **Análisis FODA**
   - Análisis personal de fortalezas, oportunidades, debilidades y amenazas

7. **Investigación de Empresa**
   - Documentación de la empresa objetivo

8. **Formulario de Contacto**
   - Envío AJAX con feedback visual
   - Template HTML profesional en los correos recibidos
   - Validación de campos

9. **Panel de Administración**
   - Gestión completa de todo el contenido
   - Interfaz intuitiva para actualizar información
